# CLAUDE.md — Wellness Checklist

Petite application React + Vite de checklist bien-être. Projet minimal : toute l'app est dans `src/App.jsx`, entrée dans `src/main.jsx`. Pas de backend.

## Commandes

```bash
npm install
npm run dev        # http://localhost:5173
npm run build      # build de prod dans dist/
npm run preview
```

## Conventions

- Seul dépôt **public** du compte GitHub (vbourgo) : ne rien y committer de personnel ou de sensible.
- Garder le projet volontairement simple — pas de dépendances supplémentaires sans demander.
